# AI Short-form Content Generator 🎥



An automated pipeline for creating high-quality short-form content (TikTok, YouTube Shorts, Instagram Reels) leveraging OpenAI's advanced models.



## 🚀 Overview

This project aims to democratize content creation by automating the most time-consuming parts of the process:

- **Trend Analysis**: Identifying viral topics from community discussions.
- 
- **Script Generation**: Crafting engaging, platform-optimized scripts using GPT-4.
- 
- **Visual Asset Creation**: Generating relevant images/backgrounds using DALL-E.
- 
- **Voiceover & Subtitles**: Integrating TTS and automated captioning.
- 


## ✨ Key Features

- 🧠 **Smart Ideation**: Automatically extracts interesting stories from various web sources.
- 
- ✍️ **Dynamic Scripting**: Adapts tone and style for different target audiences.
- 
- 🖼️ **Visual Consistency**: Ensures all generated assets align with the script's theme.
- 
- 🛠️ **Open Source**: Fully customizable and extensible for developers.
- 


## 🛠️ Installation & Usage

```bash

git clone https://github.com/dokidodo0426/ai-shortform-generator.git

cd ai-shortform-generator

pip install -r requirements.txt

python main.py --topic "Interesting Space Facts"

```
